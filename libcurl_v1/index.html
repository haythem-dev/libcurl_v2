Quick Start

Get libcurl built and installed in just a few commands:

# Download the scripts
git clone <this-repo> && cd libcurl-build-guide

# Start the web interface
python3 app.py

# Or use scripts directly
bash scripts/download_libcurl.sh
bash scripts/build_libcurl.sh

Getting Started

Web Interface
  1. Start the Flask application: python3 app.py
  2. Open your browser to http://localhost:5000
  3. Configure build options and click "Start Build"
  4. Monitor progress in real-time
Command Line
  1. Download source: bash scripts/download_libcurl.sh
  2. Build library: bash scripts/build_libcurl.sh
  3. Verify installation: bash scripts/verify_build.sh
  4. Clean up: bash scripts/cleanup.sh
